## 詐欺集團組織簡介

[你了解詐騙集團嗎? 詐欺集團的組織分工及求職陷阱](https://www.tpc.moj.gov.tw/292885/976681/661783/1239674/post)

| 角色 | 說明 |
| ---: | :--- |
| 首腦 | 整個詐騙集團的領導者，負責詐騙工作的策劃及分工。 |
| 行騙者 | 詐欺機房成員，是實際與被害人互動並騙取被害人匯款的人，方式可能包含電話、簡訊、通訊軟體等。 |
| 取簿手 | 以前往超商、置物櫃等方式，收受人頭帳戶之人。 |
| 車手 | 負責將贓款提領出來，或是向被害人面交詐欺贓款的人。 |
| 收水手 | 負責收受車手提領出來的贓款，再把錢交給上手或透過各式管道，輾轉將錢交回給集團的人。 |

## 詐欺集團組織簡介

[你了解詐騙集團嗎? 詐欺集團的組織分工及求職陷阱](https://www.tpc.moj.gov.tw/292885/976681/661783/1239674/post)

由於首腦通常隱身幕後，因此往往也是最難抓到的詐騙集團成員。

收水手的功能在於透過輾轉交付金錢的過程，增加金流上的斷點、造成警方查緝困難。

由於取簿手、車手、收水手的工作內容需要在外拋頭露面，因而也是整個詐騙集團中最容易遭查獲、取代性也最高的成員，所以也是詐欺集團成員最有可能利用圈套誘使民眾前往擔任的角色。
